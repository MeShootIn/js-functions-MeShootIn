# Тинькофф Финтех, весна 2022. Javascript.

### Установка

Установка зависимостей:

```
npm install
```

### Запуск линтера

Чтобы проверить ошибки линтера (запускается в CI):

```
npm run lint
```

Чтобы исправить ошибки:

```
npm run lint:fix
```

### Запуск тестов

Чтобы запустить тесты (запускается в CI):

```
npm run test
```
